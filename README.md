Global Layoff Trends & Industry Impact Analysis
📌 Project Overview

This project analyzes global layoff trends across industries, companies, countries, and funding stages using SQL and Power BI. The objective is to transform raw layoff data into meaningful business insights through data cleaning, exploration, and interactive dashboarding.

The project demonstrates an end-to-end analytics workflow, including:

Data Cleaning using SQL
Data Preparation and Transformation
Exploratory Data Analysis (EDA)
KPI Development
Interactive Power BI Dashboard Creation
Business Insight Generation
🎯 Business Problem

Mass layoffs have significantly impacted industries worldwide in recent years. Organizations, investors, and analysts need a better understanding of:

Which industries were most affected
Which companies experienced the largest layoffs
Countries with the highest workforce reductions
Layoff trends over time
Funding stages associated with major layoffs

This project provides a centralized dashboard to answer these questions and support data-driven decision-making.

🛠️ Tools & Technologies
SQL – Data cleaning and transformation
