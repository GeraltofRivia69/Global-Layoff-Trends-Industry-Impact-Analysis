# SQL-Layoffs-Data-Cleaning
A complete SQL-based data cleaning workflow using a global layoffs dataset. The dataset was cleaned by identifying and removing duplicate records, standardizing text values, handling null and missing data, improving overall data quality for analysis. The project highlights essential data preparation techniques commonly used in real-world analytics.

Project Overview:
This project focuses on cleaning and preparing a global layoffs dataset using SQL. The goal was to transform raw data into a clean, consistent, and analysis-ready dataset by applying industry-standard data cleaning techniques.

Tools Used:
SQL
MySQL

Data Cleaning Steps:
1)Removed duplicate records
2)Standardized company and industry names
3)Handled null and missing values
4)Corrected inconsistent entries
5)Improved overall data quality

Project Files:
Raw Dataset
Cleaned Dataset
SQL Data Cleaning Script
Objective

To demonstrate practical SQL data cleaning skills and create a reliable dataset for future exploratory data analysis and visualization.
